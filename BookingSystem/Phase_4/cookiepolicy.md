
# Cookie Policy – Booking System

_Last updated: 2025_

This Cookie Policy explains how the Booking System (“we”, “our”) uses cookies and similar technologies.

---

## 1. What Are Cookies?

Cookies are small text files stored on your device when you visit a website.  
They help provide core functionality such as:

- Maintaining login sessions  
- Remembering user preferences  
- Securing authentication  

---

## 2. Types of Cookies We Use

### 1. **Essential Cookies**
Required for system functionality:

- Authentication/login sessions  
- User role handling  
- Security validation  

These cookies cannot be disabled.

### 2. **Functional Cookies**
Improve user experience:

- Saving user preferences  
- UI settings  

Used only if applicable.

### 3. **Analytics Cookies** (Optional)
Used to improve system performance.

We do **not** run any tracking cookies unless the user consents.

---

## 3. Third-Party Cookies

Our system does not knowingly use third-party marketing cookies.  
If external providers are added in future releases, this policy will be updated.

---

## 4. Cookie Control

Users can:

- Disable cookies in browser settings  
- Delete stored cookies at any time  

Note: Disabling essential cookies may break login functionality.

---

## 5. GDPR Compliance

Cookies are used according to GDPR and ePrivacy Directive requirements:

- Only essential cookies are automatic  
- Non-essential cookies require user consent  
- Users may withdraw consent at any time
